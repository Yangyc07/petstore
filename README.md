### 该项目为一个在线的宠物商城，分为前台和后台。
### 前台功能：用户登陆注册，修改用户密码，浏览商品，商品的查询，商品详情页浏览，添加商品至购物车，查看购物车，生成订单，查看订单信息，订单的在线付款等功能，用户地址的添加，删除，更新。
### 后台功能：查看商品信息，添加商品信息。
## 项目环境
- IDE:idea 2017
- 后端框架：SpringBoot
- 前端：css/html/js,Bootstrap,thymeleaf
- 缓存:redis,Guava Cache
